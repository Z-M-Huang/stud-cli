# stud-cli

A bare-bones coding CLI tool, aiming for **100% customizability and extensibility**.

> This is a placeholder release (`0.0.1`) to claim the name on npm.
> No usable functionality is shipped yet. The architecture is still in design.

## Install

```bash
npm install -g stud-cli
```

## Usage

```bash
stud-cli            # prints a placeholder notice
stud-cli --version  # prints the current version
```
